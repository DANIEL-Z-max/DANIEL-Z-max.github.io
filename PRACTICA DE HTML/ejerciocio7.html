Lista de Definiciones

HTTP
Protocolo de transferencia de hipertexto que permite la comunicación entre clientes y servidores web.
HTML
Lenguaje de marcado utilizado para estructurar y presentar contenido en la web.
URL
Localizador uniforme de recursos; es la dirección que se utiliza para acceder a recursos en internet.
TCP/IP
Conjunto de protocolos que permiten la transmisión de datos en redes como Internet.
Internet
Red global de computadoras interconectadas que comparten información mediante protocolos comunes.
W3C
Consorcio internacional que desarrolla estándares para asegurar el crecimiento de la web a largo plazo.
Mozilla Foundation
Organización sin fines de lucro que promueve la apertura, innovación y participación en la web.
