Alguna vez te has preguntado que son los Cuadros si no es el caso pues no me importa (solo es broma profe)

Cuadros en diseño web son elementos visuales rectangulares (como cajas)que sirven para organizar y mostrar contenido dentro de una página, texto, imágenes o formularios.

Los cuadros pueden tener diferentes tamaños, colores y estilos, y se utilizan para mejorar la estructura y la estética de un sitio web.
